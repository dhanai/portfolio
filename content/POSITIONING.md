## Five interview stories (5 min each)

### 1. Takeout Order — creative fatigue meets revenue
- **Situation:** Brand growing, Meta spend scaling
- **Action:** Iterated creative, tracked ROAS/CTR, refreshed winners
- **Result:** $3.6M over 6 years; learned refresh cycles (now in Margenie Ad Ops)
- **Anthropic tie:** Brand web + performance + craft

### 2. Printer portal — B2B ops architecture
- **Situation:** Multiple 3rd-party printers, manual coordination failing
- **Action:** Built AngularJS/Firebase portal, Shopify ingest, EasyPost labels
- **Result:** Non-engineers ran daily fulfillment
- **Deep dive ready:** Order state machine, failure modes, why build vs buy
- **Anthropic tie:** Internal tooling, CMS-like structured workflows

### 3. Margenie — steerable AI
- **Situation:** Brands need Meta/Shopify ops, not black-box AI
- **Action:** Agent tools + approval cards + mechanic explanations
- **Result:** Human-in-the-loop pause/scale/creative publish
- **Anthropic tie:** Claude-powered workflows, MCP-style tools

### 4. Parfade — consumer social loop
- **Situation:** Golf coordination fragmented across texts
- **Action:** Rounds, invites, side games, light social
- **Result:** Shipped iOS app (Expo), real-time features
- **Anthropic tie:** Interactive UX, prototyping culture

### 5. Legacy → modern stack
- **Situation:** Printer app on AngularJS/Firebase still running
- **Action:** Maintained production while building Margenie on Next.js
- **Lesson:** Stack less important than reliability and operator UX
- **Anthropic tie:** Long-lived sites, pragmatic architecture

## Legacy stack talking point (printer app)

> “The fulfillment portal ran for years on AngularJS and Firebase because it was reliable and I could maintain it solo. My greenfield work is TypeScript and Next.js. I’m not precious about frameworks — I’m precise about systems that don’t break when a printer misses a click.”

## Questions to ask Anthropic

1. How does Brand Web partner with Web Platforms on shared components vs marketing-only surfaces?
2. What does the headless CMS stack look like today, and where is editorial governance headed?
3. What creative tooling has highest leverage on the team right now (generators, MCP, design system)?
4. How do you measure success for anthropic.com / claude.com — perf, conversion, brand qual?
